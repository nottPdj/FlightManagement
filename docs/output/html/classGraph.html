std::vector< std::vector< Flight > > Graph::getBestOption ( std::string  source,
int  searchFrom,
std::string  dest,
int  searchTo,
int  maxAirlines,
std::vector< std::string >  airlineCodes 
) 
@brief Gets the best flight option (or the set of options if they are equivalent), for a given source and

destination location. If specified by client the best flight option(s) have the constraint of a maximum number of airlines used. If specified by client the best flight option(s) have the constraint of using only specified airlines.

Parameters
sourceSource location
searchFromType of source
destDestination location
searchToType of dest
maxAirlinesMaximum number of airlines that can be used
airlineCodesCodes of the only airlines that can be used
Returns
Vector with the best options

Time Complexity = O(S*D*(V+E+F)), S = nº airports in (or closest) source location, D = nº airports in (or closest) destination location, V = nº airports, E = nº flights, F = total number of flights from the minimum trip(s)
